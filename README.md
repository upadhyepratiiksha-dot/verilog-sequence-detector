# Sequence Detector

## Overview

This project implements a binary Sequence Detector in Verilog HDL using a Finite State Machine (FSM). The detector identifies a predefined input sequence and generates an output upon successful detection. The design is functionally verified through simulation using a dedicated Verilog testbench.

## Features

- Finite State Machine (FSM) based design
- Detects a predefined binary sequence
- Supports overlapping/non-overlapping detection (as applicable)
- Modular RTL implementation
- Functional verification using a Verilog testbench

## Tools Used

- Verilog HDL
- Xilinx Vivado
- ModelSim
